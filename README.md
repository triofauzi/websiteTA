# README

## Installation

```bash
  1. git clone git@github.com:n0tavaliduser/freelance-16032024.git
  
  2. cd freelance-16032024

  3. composer install

  4. cp .env.example .env

  5. php artisan key:generate

  6. php artisan storage:link

  7. npm install

  8. npm run build

  9. php artisan serve
```
